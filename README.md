# scalaworld-cats
Typelevel Workshop on Cats at Scala World 2017

## Installation
- Clone the repo
- `sbt run`
- if you see the secret message you are good to go!


## Agenda-ish
- Intro
- Type Classes
- Show + Equality
- Semigroup + Monoid
- Monad Transformers
- Either vs Validated
